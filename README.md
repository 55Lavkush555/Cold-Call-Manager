# 📞 Cold Call Manager

A simple **Cold Call Lead Manager** built using **HTML, CSS, and JavaScript**.  
This tool helps freelancers and salespeople manage business leads while doing cold calls.

All data is stored in the browser using **LocalStorage**, so no backend is required.

---

## 🌐 Live Demo

🔗 https://cold-call-manager-lime.vercel.app/

---

## 🚀 Features

- ➕ Add business leads
- 📱 Store phone numbers
- 📝 Add notes for each lead
- ❌ Delete leads
- 💾 Automatic data saving using LocalStorage
- 📤 Export leads to CSV format

## 📂 How It Works

1. Enter the **Business Name**
2. Enter the **Phone Number**
3. Add optional **Notes**
4. Click **Add Lead**
5. Manage leads directly from the dashboard

All leads are stored locally in your browser.

---

## 📤 Exporting Data

You can export all leads into a **CSV file**.

This allows you to:

- Open the data in Microsoft Excel
- Use it in Google Sheets
- Create a backup of your leads

---

## ⚠️ Important Note

Since this project uses **LocalStorage**, the data is stored only in your browser.

If browser storage is cleared, the saved leads will also be removed.
